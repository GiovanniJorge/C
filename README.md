## Sobre

Este repositório reúne exercícios e projetos desenvolvidos em linguagem C, destinado a fins educacionais, estudos e aprimoramento da programação. Os exemplos abordam temas fundamentais, como manipulação de strings, utilização de subrotinas (funções) e boas práticas de codificação.

## Estrutura dos diretórios

- [`Exercícios-Strings`](https://github.com/GiovanniJorge/C/tree/main/Exerc%C3%ADcios-Strings): Exemplos e exercícios relacionados ao tratamento e manipulação de strings em C.
- [`Exercícios-Subrotina`](https://github.com/GiovanniJorge/C/tree/main/Exerc%C3%ADcios-Subrotina): Exercícios focados na criação e utilização de subrotinas/funções.

Além disso, o repositório conta com arquivos de configuração:
- `.gitignore`
- `.gitattributes`
- [LICENSE (MIT)](https://github.com/GiovanniJorge/C/blob/main/LICENSE)

## Como utilizar

1. Clone o repositório:
   ```bash
   git clone https://github.com/GiovanniJorge/C.git
   ```
2. Acesse o diretório desejado e explore os códigos fontes.
3. Compile cada exemplo utilizando o GCC:
   ```bash
   gcc exemplo.c -o exemplo
   ./exemplo
   ```
   *Substitua `exemplo.c` pelo nome do arquivo desejado.*
